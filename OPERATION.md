#### 🚀 视图拖动方式

- 鼠标中键按下并拖动
- Ctrl + 左键拖动（Windows）
- Command + 左键拖动（Mac）

#### 🚀 文本 / 段落拖动方式
- 选中文本或段落后，可直接拖拽调整位置

#### 🚀 操作流程
1. 添加所需数量的文本块和段落
2. 点击任意文本或段落，修改其基础属性
3. 点击右下角的 AI 按钮，系统会根据内容对简历进行测评（已自动过滤个人信息）
4. 点击下载按钮后，系统会按您的<strong style="color:red">段落顺序</strong>排成 A4 版式并生成 PDF。

#### 🚀 关于系统
项目所有数据均 <strong style="color:red">保存在本地</strong>，不会上传或收集任何用户信息。

#### 🚀 关于ai
本项目使用的模型：**deepseek-chat**
